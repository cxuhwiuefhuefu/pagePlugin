# 前言
#### 经常在搜索的时候会出现许多分页 发现这个翻页的按钮挺好玩的 所以想实现一款基于jQuery的翻页插件 插件是采用匿名立即执行函数 避免污染全局变量 函数尽量功能模块化 
# 功能
- [x] 已完成功能
  - [x] 点击"上一页"向上翻
  - [x] 点击"下一页"向下翻
  - [x] 点击某页翻到
# 收获 
#### 学会封装插件的思想 熟悉使用一些jQuery的方法 例如\$.fn.extend $.extend等等 代码耦合度低 功能尽量模块化
